## Response & Disclosure Process
- You can expect an **acknowledgement within 48 hours**
- We will investigate and validate the report promptly
- If accepted, a fix will be developed and released as soon as possible
- Public disclosure will occur **only after a fix is available**

If a report is declined, we will provide a clear explanation.

---

## Security Best Practices
- Never commit API keys, secrets, or credentials
- Use environment variables for sensitive configuration
- Keep dependencies up to date
- Follow least-privilege access principles
- Validate and sanitize all user inputs
